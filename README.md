# encrypt-decrypt-tool-c
Professional cryptography tool featuring 8 encryption algos: CaesarCipher(shift), VigenereCipher(keyword), ReverseCipher, AtbashCipher(mirror), XORCipher(bitwise), RSACipher(public key),&amp;Base64 Encoding. Includes encryption history tracking, RSAkey generation, strength analysis (VERYSTRONG/STRONG/MODERATE/WEAK), file-basedstorage,interactivemenu.
